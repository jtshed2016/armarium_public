About the Project

"Manuscripts of the Robbins Collection" was created in 2016 by Jordan Shedlock as a final project for the Master in Information Management and Systems degree at the UC Berkeley School of Information, advised by Robert Glushko and in cooperation with Andrea Quinn and Jennifer K. Nelson of the Robbins Collection. Robbins Collection Director Laurent Mayali, Berkeley Law Library Director Kathleen Vanden Heuvel, and the Dean of the UC Berkeley School of Law provided generous support and encouragement.

The application is based on descriptions created by the following catalogers:

  • Jennifer K. Nelson
  • Luminita Florea
  • Silvia Donghi
  • Kathleen A. Stewart
  • Alexandre M. Roberts
  • Ana E. Enriquez
  • Pamela Diaz
  • Saul M. Friedman
  • Tyler Lange
  • Jesse W. Torgerson
  • Thalia Anagnostopoulos
  • Emanuele Conte
  • Paul Hamburg
  • Nicolas Laurent-Bonne
  • Linda Ogden

The application is written in Python using the Flask framework and a SQLite database. It makes use of the Flask-SQLAlchemy, Beautiful Soup, fastpbkdf2, and WTForms Python libraries, and incorporates D3.js visualizations. Maps are embedded using the Google Maps API and location data from Geonames. Code is available on Github.
